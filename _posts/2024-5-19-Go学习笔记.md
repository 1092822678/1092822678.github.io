# Golang中`...`用法 / Go函数接受可变参数
* 第一个用法用于函数有多个不确定桉树的情况，可接受任意数量的参数
```Go
func Sum(vals ...int) int {
    total := 0
    for _, val := range vals {
        total += val
    }
    return total
}
```
* 第二个用法是将Slice打散进行传递, 即当`...`出现在切片后面的时候，表示将切片展开
```Go
nums := []int{1, 2, 3}
fmt.Println(nums...)  // 输出1 2 3
```
```Go
func test1(args ...string) { //可以接受任意个string参数
    for _, v:= range args{
        fmt.Println(v)
    }
}

func main(){
var strss= []string{
        "qwr",
        "234",
        "yui",
        "cvbc",
    }
    test1(strss...) //切片被打散传入
}
//output:
//qwr
//234
//yui
//cvbc
```
